## ✨ Features
- 🧠 Memory-based gameplay
- 🔊 Sound feedback per action
- 📊 Persistent high score (localStorage)
- 🎮 Start button (no keyboard required)
- 📱 Mobile-friendly design
- ⚡ Neon arcade UI redesign

---

## 🛠️ Tech Stack
- HTML5
- CSS3 (Neon UI styling)
- JavaScript (Game logic & DOM manipulation)
- jQuery (event handling)

---

## 🎮 How to Play
1. Click **Start Game**
2. Watch the color sequence
3. Repeat the pattern in order
4. Each level adds a new step
5. One mistake = game over
